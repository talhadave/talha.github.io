
Portfolio
Welcome to my portfolio! This repository showcases some of the projects I have worked on, highlighting my skills and expertise as a web developer. Below is a brief overview of the projects included in this portfolio:

Project 1: Daryaft (Website)
Daryaft is a Stackoverflow-like web application that I developed during a summer internship at Arbisoft. It demonstrates my proficiency in web development using Django, a Python web framework. The project features user registration, authentication, question posting, answer submission, and voting mechanisms.

Project Repository: GitHub - Daryaft
Project 2: Digital Fertilizer Recommended System
The Digital Fertilizer Recommended System is a Java console application that leverages object-oriented principles to generate personalized fertilizer recommendations based on crop and soil conditions. The project demonstrates my skills in designing flexible and extensible system architectures.

Project 3: Tic Tac Toe Game | Python with Artificial Intelligence
This project is a Tic Tac Toe game developed in Python, incorporating artificial intelligence techniques. The game includes an intelligent AI player capable of making optimal moves against human players. It showcases my understanding of AI concepts and algorithm implementation.

Feel free to explore the individual project repositories for more details and source code.
